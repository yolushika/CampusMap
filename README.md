# 🗺️ 校园地图查询导引系统

> 数据结构课程设计 · 东北石油大学 · 计算机与信息技术学院

基于 **C++** 的控制台校园地图导航软件，采用**邻接矩阵**存储图结构，使用 **Dijkstra 算法** 计算最短路径，支持路径导引、距离查询、最近地点查询和地图数据维护。

---

## 功能一览

| 主菜单 | 功能 | 说明 |
|--------|------|------|
| 1 | 📂 调用地图 | 从文本文件加载校园地图数据 |
| 2 | 📋 显示全部地点 | 列出所有已加载的地点名称 |
| 3 | 📏 距离查询 | 查询从某地点到所有地点的最短距离 |
| 4 | 🧭 路径导引 | 两点间最短路径 + 转向提示（直行/左转/右转） |
| 5 | 📍 最近景点查询 | 查找最近的 N 个地点 |
| 6 | 🔧 地图数据维护 | 二级菜单：增删改地点和道路 |
| 7 | 💾 保存地图 | 将当前地图保存到文件 |
| 0 | 🚪 退出系统 | — |

### 地图数据维护子功能

- 增加 / 删除 / 修改地点（顶点）
- 增加 / 删除 / 修改道路（边），支持保留原值的灵活修改

## 核心算法

**Dijkstra 最短路径算法** — O(n²) 时间复杂度

```
1. 初始化：所有顶点标记为"未确定"，距离设为直接边距离
2. 贪心选择：在未确定顶点中选距离最小的
3. 松弛操作：用新确定的顶点更新其邻居的距离
4. 重复 2-3 直到所有顶点确定
```

路径导引中，从 `P[]` 前驱数组回溯构建完整路径，通过方向角差值计算转向提示。

## 构建与运行

### 环境要求

- **Visual Studio 2026**（或 2022+）
- MSVC 编译器（v145 工具集）
- Windows 10+

### 编译

在 VS2026 中打开 `CampusMap.slnx`，按 `Ctrl+Shift+B` 或选择 **生成 → 生成解决方案**。

或使用命令行：

```bash
MSBuild CampusMap.slnx /t:Build /p:Configuration=Debug /p:Platform=x64
```

### 运行

编译后在 VS2026 中按 `F5` 运行，或直接执行：

```bash
x64\Debug\CampusMap.exe
```

按提示加载地图文件（如 `map.txt`）即可开始使用。

## 设计要点

- **邻接矩阵选型理由**：校园地图顶点数 ≤30，邻接矩阵 O(1) 边查询 + 实现简洁
- **无向图 / 有向图切换**：通过 `GRAPH_TYPE` 宏在编译期切换，零运行时开销
- **删除顶点**：采用"覆盖法"（用最后一个顶点填充被删位置），避免移动大量元素
- **修改道路**：输入 `-` / `-1` 保留原值，无需重复输入
- **编码兼容**：支持 UTF-8 和 GBK 编码的地图文件
